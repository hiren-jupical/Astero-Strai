By default in Odoo, when you resize the column width on list view or one2Many field, this setting only keep your current session, if you re-login again or access to Odoo with another browser, the column width will apply Odoo default.
With this app, will help you save column with setting, even you re-login or access to Odoo with another browser, your column width setting still keep from last setting.


For example. I open the order list on different browsers, Edge and Chrome. If I change the column width in the Edge browser, then switch to the Chrome browser and simply reload the order list screen, the column width setting will be the same as in the Edge browser.

And this is an example on the sale order line (One2Many field).
