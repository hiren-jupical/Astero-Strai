Flexible Chatter

Ability to smoothly change the width of the Chatter panel

Move the mouse cursor to the edge of the Chatter and drag with the left mouse button to change the width of the panel

Ability to quickly hide or show the Chatter panel

When you hover mouse over the Chatter panel, a button will appear that allows you to show or hide the panel

Ability to move the Chatter panel down

In your profile settings select the bottom Chatter position

Settings

Ability to change the Chatter panel position for each user

In your profile settings select the Chatter page and then the Chatter position
